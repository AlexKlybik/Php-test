Ошибка/Исключение               Причина возникновения                                   Поведение программы
Incorrect URL                   Отсутствует указание метода API                         Возвращает сообщение с ошибкой
Not valid method name           Имя метода API указано неверно                          Возвращает сообщение с ошибкой и указанием примера наименования метода
Not found                       Указан метод API для которого отсутствует реализация    Возвращает сообщение с ошибкой с указанием отсутствующего метода
Server error                    Ошибка по независящим от пользователя причинам          Ошибка добавляется в лог и возвращается сообщение "Server error"
Incorrect parameters            Если не переданы необходимые параметры                  Возвращается сообщение с ошибкой с указанием необходимых параметров
Data not found                  По заданным параметрам данные не найдены                Возвращает сообщение с ошибкой "Data not found"
